# Autotests Project
Проект содержит автоматизированные UI- и API-тесты для веб-приложения книжного магазина и внешнего API.  
Тесты реализованы с использованием **Python**, **Pytest**, **Selenium WebDriver**, **Requests** и **Allure**.

## Описание задачи

В рамках задания необходимо:
- реализовать UI- и API-автотесты по функциональному чек-листу
- покрыть основные пользовательские сценарии (поиск, корзина, работа с API)
- использовать Page Object
- обеспечить возможность запуска тестов
